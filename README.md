# Asma'ul Husna Learning App for Grade 5

An interactive educational web application designed to help 10-year-old students learn and test their knowledge of Allah's 99 names (Asma'ul Husna).

## About

This project provides an engaging way for primary school students to learn the beautiful names of Allah through interactive matching quizzes that connect Arabic names with their Malay meanings.

## Features

- **Educational Introduction**: Brief notes about Asma'ul Husna suitable for young learners
- **Interactive Quiz**: Matching game to test knowledge by connecting Arabic names with Malay meanings
- **Child-Friendly Design**: Large fonts, colorful interface, and simple navigation optimized for 10-year-old users

## Project Structure
